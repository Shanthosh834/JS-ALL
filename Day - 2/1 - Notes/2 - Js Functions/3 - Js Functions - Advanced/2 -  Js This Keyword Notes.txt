
9. Js This Keyword

    * this = refers to the CURRENT object that is executing the function

    * SIMPLE: "this" means "the owner of this function"

    * 4 SITUATIONS OF this:

        * 1. this in Global Scope

            javascriptconsole.log(this); 
            // In browser → window object
            // this = window (global object)

        * 2. this in Object Method

            const person = {
                name: "Shanthosh",
                greet: function() {
                    console.log(this.name); // "Shanthosh"
                    // this = person object
                }
            };

            person.greet(); // Shanthosh ✅

        * 3. this in Regular Function

            javascriptfunction show() {
                console.log(this);
                // In browser → window object
                // this = global window
            }

            show();

        * 4. this in Arrow Function

            javascriptconst person = {
                name: "Shanthosh",

                // Regular function - this works!
                greet: function() {
                    console.log(this.name); // "Shanthosh" ✅
                },

                // Arrow function - this DOESN'T work!
                greetArrow: () => {
                    console.log(this.name); // undefined ❌
                    // Arrow has NO own this!
                    // Takes this from parent scope (window)
                }
            };

            person.greet();      // Shanthosh ✅
            person.greetArrow(); // undefined ❌

    * KEY DIFFERENCE:

    * Regular function:

        ✅ HAS its own this
        ✅ this = whoever calls the function

    * Arrow function:

        ❌ NO own this
        ❌ Takes this from parent scope

    * = USE REGULAR FUNCTION when you need this!

    * = USE ARROW FUNCTION when you don't need this!

    * IMPLE EXAMPLE TO REMEMBER:

        javascriptconst student = {
            name: "Shanthosh",
            age: 23,

            // this works here!
            getInfo: function() {
                return `${this.name} is ${this.age}`;
            }
        };

        console.log(student.getInfo()); 
        // "Shanthosh is 23" ✅
        // this = student object
