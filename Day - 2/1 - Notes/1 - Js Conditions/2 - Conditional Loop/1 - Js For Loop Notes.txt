* Loops can execute a block of code a number of times

* There are 5 Types:

    1. Js For Loop
    2. Js While Loop
    3. Js Break
    4. Js Continue
    5. Js Control Flow

1. Js For Loop

    * For Loop can execute a block of code a number of times.

    * Syntax:
        for (exp 1; exp 2; exp 3) {
          // code block to be executed
        }

    * Loop Scope:

    * use let inside the for loop

        * let i = 5;

            for (let i = 0; i < 10; i++) {
              // some code
            }

            // Here i is 5 

        * if we use var its not corrrect

            var i = 5;

            for (var i = 0; i < 10; i++) {
              // some code
            }

            // Here i is 10
